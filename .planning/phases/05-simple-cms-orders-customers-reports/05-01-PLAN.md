Create supabase/schema.sql with:
1. **customers** table: id (uuid), email (unique), name, phone, created_at
2. **orders** table: id (uuid), order_id (text, format ORD-YYYYMMDD-XXXX per D-19), customer_id (FK), customer_name, customer_phone, customer_address, total, status (pending/confirmed/delivered), created_at
3. **order_items** table: id (uuid), order_id (FK), product_id, product_name, unit_price, quantity, subtotal
4. Indexes: customer_id on orders, order_id on order_items, created_at for time queries
5. RLS: Enable on all tables
   - customers: Users can read own data, admins can read all
   - orders: Users can read own, admins can read all; insert for authenticated
   - order_items: Same as orders

Use PostgreSQL UUID generation: gen_random_uuid()
Use proper foreign key constraints with ON DELETE CASCADE
  </action>
